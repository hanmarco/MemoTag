# MemoTag

Windows에서 현재 활성 앱 창에 색상 테두리와 메모를 남기는 작은 WPF 앱입니다.

## 사용법

- 실행: `dotnet run --project MemoTag`
- 현재 창에 표시 추가/수정: `Ctrl+Alt+M`
- 현재 창 표시 해제: `Ctrl+Alt+Shift+M`
- 트레이 아이콘 우클릭: 현재 창 표시, 현재 창 해제, 전체 해제, 종료

표시는 현재 Windows 세션에서 열린 창 핸들에 붙습니다. 대상 창을 이동하거나 크기를 바꾸면 테두리와 메모도 따라가고, 대상 창이 닫히면 표시도 자동으로 제거됩니다.

## 현재 MVP 범위

- 현재 열려 있는 앱 창 기준으로 동작합니다.
- 앱 재시작 후 이전 메모를 자동 복원하지는 않습니다.
- 관리자 권한으로 실행 중인 앱 위에 표시하려면 이 앱도 관리자 권한으로 실행해야 할 수 있습니다.
